Writing a Jenkins plugin

Jenkins is a powerful open source continuous integration tools that enables you to automatically build and test your software projects (among many other things). If you use Jenkins, you surely have already installed a lot of plugins that answer more or less to your needs. But have you ever tried to develop your own plugin?

This note provides a quick sum up of the basic steps to write your own Jenkins plugin.
